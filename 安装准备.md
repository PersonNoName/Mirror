# 安装准备清单

## 一、必须安装（核心依赖）

### 1.1 Python 环境

```bash
# 推荐 Python 3.11+
python --version  # 确保 >= 3.11
```

---

### 1.2 核心运行时依赖

```bash
pip install pydantic>=2.0
pip install httpx>=2.28
pip install httpx-sse>=0.4
pip install asyncio
```

---

### 1.3 开发工具

```bash
pip install ruff>=0.1    # lint 和格式化
pip install pytest>=7.4  # 测试（可选，本项目用自定义测试框架）
```

---

## 二、存储层依赖

### 2.1 Redis（推荐，用于 TaskStore + CoreMemoryStore）

```bash
# macOS
brew install redis
brew services start redis

# Ubuntu/Debian
sudo apt install redis-server
sudo systemctl start redis

# Windows (WSL2 建议)
# 或使用 Docker
docker run -d -p 6379:6379 redis:latest
```

**Python 客户端**：

```bash
pip install redis>=5.0
pip install hiredis>=2.0  # 可选，C扩展加速
```

---

### 2.2 PostgreSQL（可选，用于 JournalStore + SnapshotStore）

```bash
# macOS
brew install postgresql@16
brew services start postgresql@16

# Ubuntu/Debian
sudo apt install postgresql postgresql-contrib
sudo systemctl start postgresql

# Docker
docker run -d -p 5432:5432 -e POSTGRES_PASSWORD=secret postgres:16
```

**Python 客户端**：

```bash
pip install asyncpg>=0.29   # 异步 PostgreSQL 客户端
pip install databases>=0.9  # 异步 SQLAlchemy 替代
# 或
pip install sqlalchemy[asyncio]>=2.0
pip install psycopg>=3.1
```

### Docker Compose（启动所有基础设施）

```yaml
# docker-compose.yml
version: '3.8'

services:
  redis:
    image: redis:latest
    ports:
      - "6379:6379"
  
  postgres:
    image: postgres:16
    environment:
      POSTGRES_PASSWORD: secret
    ports:
      - "5432:5432"
  
  qdrant:
    image: qdrant/qdrant:latest
    ports:
      - "6333:6333"
      - "6334:6334"
  
  neo4j:
    image: neo4j:latest
    environment:
      NEO4J_AUTH: neo4j/secret
    ports:
      - "7474:7474"
      - "7687:7687"
```

**启动**：

```bash
docker-compose up -d
```

## 九、验证安装

```bash
# 验证 Python 依赖
python -c "import pydantic, httpx, fastapi, redis, openai, qdrant_client, neo4j; print('All imports OK')"

# 验证 Redis
redis-cli ping  # 应返回 PONG

# 验证 PostgreSQL
psql -U postgres -c "SELECT version();"  # 应返回版本信息

# 验证 Qdrant
curl http://localhost:6333/readyz  # 应返回 {"status":"ok"}

# 验证 Neo4j
curl http://localhost:7474 -H "Accept: application/json" -u neo4j/secret  # 应返回 API 信息
```

## 十一、推荐开发环境

```bash
# IDE: VS Code 或 Cursor
# Python 插件: Python, Pylance, Ruff

# 虚拟环境

source venv/bin/activate  # Linux/macOS
```
