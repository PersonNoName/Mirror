# Source Encoding Rule

- all source files and markdown files in this repository must be saved as UTF-8
- do not commit mojibake, mixed encodings, or editor-specific legacy encodings
- when text corruption is detected, fix the file before adding more content
- for machine-facing prompts, prefer stable English unless another language is explicitly required
- for user-facing multilingual text, verify the file is saved as UTF-8 before commit
